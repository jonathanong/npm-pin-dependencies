Pins all your `package.json`'s `dependencies`, `devDependencies`, and `optionalDependencies`.
Designed to be used with http://greenkeeper.io/.

Usage:

```bash
npm i -g npm-pin-dependencies
cd my-app
npm-pin-dependencies
```
